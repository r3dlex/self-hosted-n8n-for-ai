# Contributing

See [AGENTS.md](AGENTS.md) for project governance, conventions, and architecture decisions.

## Quick guidelines

- **Small PRs only** — one feature or fix per PR
- **Test your changes** — run `docker compose up -d` and verify services start cleanly
- **Keep it simple** — this is a learning/experimentation kit, not production infrastructure
- Typo-only PRs will be rejected
